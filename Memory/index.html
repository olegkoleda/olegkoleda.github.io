Welcome into the memory game

The memory game is a basic matching game to test the player's memory. In a deck of paired cards, the player needs to match each pair to win the game.

Rules:

  1. Select two cards to try match the pictures.
  2. If you match the pictures you can go again.
  3. If they don't match it is computer turn them back.
  4. The player that finds all pairs wins!
  5. GLHF.
